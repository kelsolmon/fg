Guía Básica de CSS

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. CSS controla cómo los elementos HTML deben ser renderizados en la pantalla, en papel, o en otros medios.

2. Funcionamiento Básico de CSS

CSS se basa en reglas que definen el estilo de los elementos HTML. Estas reglas consisten en un selector y un bloque de declaración. El selector apunta al elemento HTML que deseas estilizar y el bloque de declaración contiene una o más declaraciones separadas por punto y coma. Cada declaración incluye una propiedad y un valor.

3. Incluir CSS en un archivo HTML

CSS se puede incluir en un archivo HTML de tres maneras:

  • CSS en línea: Se utiliza el atributo style dentro de un elemento HTML. Ejemplo: <p style="color: blue;">Texto azul</p>
  • CSS interno: Se utiliza la etiqueta <style> dentro de la sección <head> del documento HTML.
  • CSS externo: Se enlaza un archivo CSS externo usando la etiqueta <link> dentro de la sección <head>.

4. Definición de una regla CSS

Una regla CSS consiste en un selector y un bloque de declaración. Aquí tienes un ejemplo básico:


/* Ejemplo de una regla CSS */
p {
    color: red;
    font-size: 16px;
}

5. Elementos de una regla CSS

  • Regla: Es la combinación de un selector y un bloque de declaración.
  • Selector: Apunta al elemento HTML que deseas estilizar. Ejemplo: p en la regla anterior selecciona todos los elementos <p>.
  • Bloque de declaración: Contiene una o más declaraciones CSS. Está delimitado por llaves { }.
  • Propiedad: Especifica qué aspecto del elemento HTML deseas cambiar. Ejemplo: color, font-size.
  • Valor: Define el valor de la propiedad. Ejemplo: red, 16px.

6. Medios CSS

Los medios CSS se refieren a la capacidad de aplicar diferentes estilos según el dispositivo o las características de la pantalla. Esto se logra utilizando las consultas de medios (@media). Ejemplo:


/* Aplica estilos solo en pantallas con un ancho máximo de 600px */
@media screen and (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

7. Comentarios CSS

Los comentarios en CSS se utilizan para dejar notas en el código y no afectan el estilo de los elementos. Se encierran entre /* y */. Ejemplo:


/* Este es un comentario en CSS */
body {
    background-color: white; /* El color de fondo es blanco */
}

8. Sintaxis de la definición de las propiedades CSS

La sintaxis básica de una propiedad CSS es:

selector {
    propiedad: valor;
}

Ejemplo:

p {
    color: green;
}
